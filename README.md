## ¿Qué hace?

1. Abres una imagen (PNG/JPG).
2. Ajustas fondo, escala, centrado / stretch / zoom y bordes.
3. Previsualizas máscaras **iOS** y **Android**.
4. Exportas el paquete listo para Xcode / Android Studio.
